## Завдання для самостійного виконання
- Завдання 1.
```
Написати програму, яка демонструє основні дії по роботі із двозв’язним списком цілих чисел: 
- додати елемент до списку (в кінець);
- вилучити вказаний елемент зі списку;
- визначити кількість елементів у списку;
- поміняти два елементи місцями, які є сусідами елемента p.
```
- Завдання 2. 
```
Реалізувати програму для роботи з двома списками:
- одержати доступ до k-го елемента двох списків, якщо вони різні – поміняти місцями в списках;
- об'єднати два лінійних списки в один список;
- розбити лінійний список на два списки;
- зробити копію лінійного списку.
```
- Завдання 3. 
```
Написати програму, яка передбачає введення в стек цілих чисел з клавіатури. 
Вивести стек на екран. Створити і роздрукувати два стеки – з парними та непарними числами 
серед введених.
```
- Завдання 4. 
```
Операції над множинами. Реалізувати операції над множинами, що задані у вигляді масивів.
Операції – об’єднання, перетин, доповнення, різниця, симетрична різниця.
```
- Завдання 5.
```
Дано рядок символів, що містить дужки різних видів (круглі, фігурні, квадратні). 
Перевірте правильність розставлення в ньому круглих дужок, будь-яких дужок.
Наприклад, [ { ( } ) { ] [ } ( [ ] ) { }.
```
- Завдання 6.
```
Реалізуйте алгоритми обчислення арифметичних виразів – перетворення в
постфіксну форму, обчислення постфіксного виразу.
Варіант 10. 2+(3+2*(5-4)/2)+3
```
- Завдання 7. 
```
 Черги часто використовуються для моделювання потоку людей, машин, банківських операцій і т.д.
Напишіть програму, що моделює чергу покупців до каси в магазині. Програма повинна відображати
вміст відразу декількох черг. Новий покупець переміщується в чергу натисканням клавіші. Ви 
повинні самостійно визначити,яким чином він буде вибирати чергу (мінімальна довжина черги).
Обслуговування кожного покупця має випадкову тривалість (в залежності від
кількості товарів в кошику). Обслужені покупці видаляються з черги.
```
